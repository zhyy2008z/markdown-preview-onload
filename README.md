# markdown-preview-onload README

在 vscode startup 后，立刻使所有处于打开状态的markdown文件变为preview模式。

## Features

该扩展用来解决 \*.md 的打开方式设置为 vscode.markdown.preview.editor 后，直接打开 markdown 文件时，打开方式未生效的问题。

## Acknowledgement

部分代码参考了扩展 prefer-preview-markdown 源码，感谢 @HugoMcPhee！
